# OP-T158 Email Template Compare Proof - 2026-06-29

## Scope

This proof covers the staff-only, provider-neutral compare follow-up on top of the existing OP-T158
saved email template drafts, preview snapshots, and immutable published-version history.

Shipped behavior:

- A pure domain helper compares a persisted saved email template draft with one immutable published
  version for the same firm and draft.
- The staff dashboard email templates panel defaults the compare target to the newest loaded
  published version and lets staff select another loaded published version.
- The compare surface renders changed/same fields for saved draft content versus published version
  content as escaped text only.

Preserved boundaries:

- `/api/email/previews` remains render-only.
- No API route, route authorization manifest, route catalog, schema, migration, database
  repository, provider setting, provider delivery, campaign automation, bulk send, subscription,
  queue, send job, audit-event write, preview-snapshot write, or outbox behavior was added or
  changed.
- Tests and examples use synthetic template data only.

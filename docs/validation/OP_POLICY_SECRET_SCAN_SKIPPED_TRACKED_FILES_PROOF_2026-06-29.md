# Policy Secret Scan Skip Gate Proof - 2026-06-29

## Scope

This proof covers the local policy-gate tightening that makes `pnpm policy:check` run the
tracked-secret scanner with `--fail-on-skipped` because tracked files that the scanner cannot
inspect should fail the everyday policy gate instead of remaining warnings.

The change preserves the existing high-confidence secret regexes, redacted report shape, explicit
path scan behavior, ignored-evidence defaults, and synthetic false-positive posture. It does not
change `.gitleaksignore`, scan ignored local evidence by default, add dependency or provider
surface area, or serialize matched secret values.

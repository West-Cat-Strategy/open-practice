## Combined Security Posture

  fields stay out of `NewAuditEvent`.

- Broad staff aggregates require `STAFF_ACCESS_REQUIRED` for `client_external` callers.
- Production authenticated CORS is limited to configured web origins.
- Inbound email promotion no longer automatically queues OCR.
- OCR queueing is rejected until the document scan state passes.
- Storage-key payloads are redacted from API, queue, and audit-visible responses.
- Docker/service image updates from the full-scan lane are preserved with the documented residual

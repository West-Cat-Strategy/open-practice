# OP-T96 Privacy And Authorization Hardening Proof

## Scope

- Redact direct audit API metadata so list and export responses expose metadata keys and taxonomy summaries, not raw event metadata.
- Scope `/api/overview` for matter-limited users to assigned matters, own user profile, and derived counts only.
- Filter inbound email status for matter-limited users and hide provider configuration from that projection.
- Paginate `/api/jobs` across invisible records so assigned-matter jobs are not hidden behind newer firm-wide or unassigned jobs.
- Minimize connector outbox `payloadSummary` to bounded allowlisted fields before external delivery.
- Add OP-T92 structured-intake diagnostics and blocked save affordances for invalid builder state.
- Add OP-T95 local release evidence and dependency-gate selection for package manifest and lockfile changes.
